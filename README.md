# Invictus TDEE & Fat Loss Calculator

Local dev (Node + npm required):

```bash
npm install
npm run dev
```

Open the URL shown by Vite (usually `http://localhost:5173`).

What you get:
- TDEE computed via Mifflin–St Jeor
- Recommended calorie range for fat loss (15–25% deficit)
- Protein guidance (1.6–2.2 g/kg)
- Practical coaching copy, insights, and a CTA to request a personalised plan
